### 本地仓库同步到github
- 本地创建仓库（或者叫项目/存档），然后在 vscode 中使用 git 工具推送到远程服务器 github
- github-Settings-可修改仓库名字
-  克隆项目：`git clone https://github.com/joyehuang/minimind-notes.git`
- 本地仓库目录下运行 `git log` 可查看历史提交
